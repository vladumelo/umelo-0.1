Сайт Умело

Структура:
- index.html — главная
- calculators.html — каталог калькуляторов
- calc-estimate.html — быстрая смета
- calc-map.html — калькулятор на карте
- style.css — общие стили
- script.js — общие скрипты
- calculators.js — логика таблицы сметы
- map-calc.js — логика карты
- assets/favicon.svg — иконка

Как открыть:
1. Распакуйте архив.
2. Откройте index.html или сразу нужную страницу калькулятора.
3. Для calc-map.html интернет нужен, потому что карта и плагины грузятся по CDN.

Что изменено:
- калькуляторы вынесены на отдельные страницы
- карта работает без API-ключа
- есть поиск участка, рисование, редактирование, экспорт CSV и сохранение JSON


Обновление карты:
- calc-map.html переведён на Яндекс-подложку через тайловый слой.
- Добавлены публичные кадастровые слои Росреестра/ПКК через ArcGIS MapServer.
- Добавлен экспорт контуров в DXF (координаты Web Mercator, метры).
- Если публичный кадастровый слой временно не отвечает, базовая карта и локальные расчёты продолжают работать.
